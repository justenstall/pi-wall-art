# pi-wall-art

Application to power wall art run on a Raspberry Pi

Experimenting currently

## Available Pinouts

Available on Adafruit Bonnet:

- SCL
- SDA
- RX
- TX
- #25
- MOSI
- MISO
- SCLK
- CE0
- CE1
- #19

Pin 24 is taken up by the 1/32 scan (64x64 matrix)

Pin 18 is taken up by the "quality" setting.
